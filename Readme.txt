Prerequisites:-

1. Need MacBook/Mac.

2. Download jar files and Configure the same in local build path. Below are list of jars.
   a) jackson-core-asl-1.9.13.jar
   b) jackson-mapper-asl-1.9.13.jar
   c) jackson-xc-1.9.13.jar
   d) json-simple-1.1.jar
   
3. Use any IDE or TextEditor. Eclipse IDE is optional. If you have, its great !!! 



Steps to RUN from Eclipse :-

  1) Open this project in eclipse 
  2) Open DarkSkyApiTest.java file
  3) Mouse right click on this file, run as Java application.
  4) Observe the output in the console.
  5) Based on this output, you can validate the data.
  
  FYI :: Due to time bound, I used only basic Json-simple way. We can handle this by Jackson too.
